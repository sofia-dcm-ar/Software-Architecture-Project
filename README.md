# Software-Architecture
Final project of "Programming Methodology I", Informatic Engineering course.  
Different Software Architectures for POO applied in C# using SharpDevelop version 5.1.0, migrated to Visual Studio 2022  

### *Key Skills*
* *Organization*
* *GitHub branches*
* *Clean Code*
* *Interface design*
* *Software Architecture design*
* *Problem solving*
* *XML Documentation*

## Project Description
* Acquire knowledge of a new software architecture every week.
* Implement each architecture in the existing software by following specific instructions and weekly exercises, with full creative and structural freedom.
* Submit weekly assignments focusing exclusively on the previous and new classes required to fulfill the objectives.

## Project Organization:     
* This repository follows an incremental development process, each week's assignment is merged into the main branch.
* To view the state of the code for a specific week, please refer to the Git Tags or the Releases section (e.g., v1.0 for Week 1, v2.0 for Week 2).
* The project uses XML Documentation for better readability. Hover over any method to see its design pattern purpose.

## Project Content
**Week 1:** Experience with interface design and implementation  
**Week 2:** Strategy pattern ; Iterator pattern  
**Week 3:** Factory Method pattern ; Observer pattern  
**Week 4:** Adapter pattern ; Decorator pattern  
**Week 5:** Proxy pattern ; Command pattern  
**Week 6:** Composite pattern ; Template Method pattern  
**Week 7:** Chain of Responsability pattern ; Singletone pattern
